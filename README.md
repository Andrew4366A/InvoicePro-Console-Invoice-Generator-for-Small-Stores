# 🧾 InvoicePro – Console Invoice Generator

## 📌 Overview
InvoicePro is a Python-based console application designed to help small store owners manage products and generate invoices efficiently.  
The system allows users to maintain product inventory, create invoices, calculate totals with tax, and save bills as text files.

---

## 🚀 Features

- 📦 Add, view, and manage products
- 🧾 Create invoices with multiple items
- 🔢 Automatic calculation of subtotal, tax, and total
- 💾 Save invoices as `.txt` files
- 📂 Load and store product data using JSON
- ⚠️ Input validation for better reliability
- 🧮 Stock management with quantity updates

---

## 🛠 Tech Stack

- Python (Core)
- Object-Oriented Programming (OOP)
- File Handling (JSON & TXT)
- CLI (Command Line Interface)

## 💡 Example Workflow

1. Add products (ID, name, price, stock)
2. View available products
3. Create invoice
4. Add items with quantity
5. Generate and save invoice
